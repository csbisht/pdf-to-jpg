# Contributing to pdf-to-jpg

Thank you for your interest in contributing! 🎉  
This is a small open-source Python utility and all contributions are welcome.

---

## Ways to contribute

- 🐛 **Report a bug** — open an [Issue](https://github.com/csbisht/pdf-to-jpg/issues)
- 💡 **Suggest a feature** — open an [Issue](https://github.com/csbisht/pdf-to-jpg/issues) with the label `enhancement`
- 🔧 **Fix a bug or add a feature** — open a Pull Request
- 📖 **Improve the docs** — fix typos, clarify steps, add examples

---

## How to submit a Pull Request

1. **Fork** the repo and clone your fork:
   ```bash
   git clone https://github.com/<your-username>/pdf-to-jpg.git
   cd pdf-to-jpg
   ```

2. **Create a branch** for your change:
   ```bash
   git checkout -b fix/your-fix-name
   ```

3. **Set up the environment** and install dependencies:
   ```bash
   python -m venv myenv
   source myenv/bin/activate      # Linux / macOS
   myenv\Scripts\Activate.ps1     # Windows PowerShell
   pip install -r requirements.txt
   ```

4. **Make your changes** and test them manually with a real PDF.

5. **Commit** with a clear message:
   ```bash
   git commit -m "Fix: describe what you fixed"
   ```

6. **Push** and open a Pull Request against the `main` branch.

---

## Code style

- Follow existing code style (clean, readable Python)
- Add docstrings to any new functions
- Keep changes focused — one fix or feature per PR

---

## Questions?

Open an [Issue](https://github.com/csbisht/pdf-to-jpg/issues) and ask — happy to help!
